## 快速上手

### 项目克隆`git clone`或者直接Download

```Bash
git clone https://github.com/ChenJiaH/vue-cnode.git
```

### 安装依赖

``` bash
npm install
```

### 运行

``` bash
npm run dev
```

### 发布

``` bash
npm run dev
```
